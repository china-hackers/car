## 登录及权限验证操作

### 1. 接口返回说明

````angular2html
300：未登录，需要前端直接跳转到登录页面进行登录
301：用户名或密码错误
302：没有该权限，禁止访问

````

### 1.1 登录
### 请求地址: /admin/login

#### 接口类型：POST

#### POST 参数

```
{
    admin:username,
    pswd:123456
}
```

### 返回数据

```
{
    code: 200,
    data: [],
    message:'错误代码对应信息，无错误返"操作成功"',
}
```
### 1.2 退出
### 请求地址: /admin/logout

#### 接口类型：GET

#### 该接口是全站唯一特殊接口

```
前端直接跳转到该地址就可以了/admin/logout，可以直接用A标签加链接的方式
```

### 返回数据

```
后台清除cookie及session之后会跳转到登录页面/admin/login
```
