### 1. 首页搜索接口
### 请求地址: /mobile/product/list

#### 接口类型：POST

#### POST 参数

```
{
    p:1 //除了p以外，其他都是可选
    brand:品牌
    shoufu_from:30000
    shoufu_to:50000
    yuegong_from:0
    yuegong_to:1000
    price_from:0
    price_to:50000
    displacement_from: 0
    displacement_to: 1.5
    gear_box:手动
    air_in:罗轮增压
}
```
