### 2 通过字母获取所有品牌
### 请求地址: /admin/car/brandlist

#### 接口类型：POST

#### POST 参数

```
{
    letter:'Z',//字母必须是大写，如果不传字母，则返回所有品牌

}
```

### 4 添加新车型
   ### 请求地址：/admin/car/add
   
   #### 接口类型：POST
   #### POST 参数
   
```$xslt
{
    letter:'B',//必须大写
    brand:'宝马',
    model:'X5',
    star:4.23,//必须是数字，小数点后2位
    keywords:'
    空间大
    推背感强
    省油
    '//关键词描述，回车来分隔即可
}
```
### 返回数据

```
{
    code: 200,
    message:'错误代码对应信息，无错误返"操作成功"',
}
```

```$xslt
    model:'X5',
    star:4.23,//必须是数字，小数点后2位
```
