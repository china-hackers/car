### 4 添加新车型
   ### 请求地址：/admin/car/add
   
   #### 接口类型：POST
   #### POST 参数
   
```$xslt
{
    letter:'B',//必须大写
    brand:'宝马',
    model:'X5',
    star:4.23,//必须是数字，小数点后2位
    keywords:'
    空间大
    推背感强
    省油
    '//关键词描述，回车来分隔即可
}
```
### 返回数据

```
{
    code: 200,
    message:'错误代码对应信息，无错误返"操作成功"',
}
```

### 7 添加/编辑二手车基本信息

   #### 接口类型：POST
   #### POST 参数就是上述基本信息的那些字段
   #### 不提交"id"这个字段或者提交空/0表示添加，提交大于0表示编辑该商品信息
