### 2 通过字母获取所有品牌
### 请求地址: /admin/car/brandlist

#### 接口类型：POST

#### POST 参数

```
{
    letter:'Z',//字母必须是大写，如果不传字母，则返回所有品牌
    p:1
}
```

### 返回数据

                "id": 2007,
                "letter": "Z",

### 2.1 编辑管理员
   ### 请求地址：/admin/role/edit
   
   #### 接口类型：POST
   #### POST 参数
   
```$xslt
{
    id:角色ID,
    role:'角色名',
    permission:[
        1,2,45,35,64,57,2,13 //权限ID，通过权限列表来获取
    ]
}
```
### 2.2 删除角色
   ### 请求地址：/admin/role/delete
   
   #### 接口类型：POST
   #### POST 参数
   
   ```$xslt
   {
       id:角色ID
   }
   ```

### 返回数据

```
{
    code: 200,
    message:'错误代码对应信息，无错误返"操作成功"',
}
```
