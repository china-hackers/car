### 1.1 获取城市信息
### 请求地址: /mobile/city/city

#### 接口类型：POST

#### POST 参数

```
{
    id:101
}
```

### 返回数据

```
{
    "code": 200,
    "data": {
        "id": 101,
        "parent_id": 100,
        "name": "合肥",
        "sort": 1
    },
    "message": "操作成功"
}
```
