## 车商管理相关接口

### 1. 获取车商列表
### 请求地址: /admin/business/list

#### 接口类型：POST

#### POST 参数

```
{
    p:1,
    check:1//0表示获取未审核的车商，1表示已审核，2表示所有车商
}
```
